## FishBot

# Features:
- Casts rod and reels rod in when fish is caught (v1)
- Avoids AFK kick by moving character and mouse position (v2)

# First time setup
- Make sure Python 3.x is installed
- Run install.bat

# To use
- Go to fishing location
- Pull out rod
- Run main.py
- Go back to game

# To stop
- Hit ESC in game
- Go to CMD and hit CTRL+C
