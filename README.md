## FishBot

First time setup:
- Make sure Python 3.x is installed
- Run install.bat

To use:
- Go to fishing location
- Pull out rod
- Run main.py
- Go back to game

To stop:
- Hit ESC in game
- Go to CMD and hit CTRL+C

Features:
- Able to fish and
- Avoids AFK by moving character and mouse position
